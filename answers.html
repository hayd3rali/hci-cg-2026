Part C: Short Answer

  1. Explain the difference between a semantic element (e.g., nav) and a generic element (e.g., div).

    The primary difference lies in meaning: a semantic element (like <nav>) explicitly describes its content's purpose to the browser and assistive technologies, which improves accessibility and SEO. Conversely, a generic element (like <div>) is a non-descriptive container used strictly for grouping content for layout and styling without conveying any structural meaning.

  2. Why does KLM not model high-level planning and interpretation?

    KLM excludes high-level planning because it is specifically designed to model expert performance on routine tasks where the user already knows the exact sequence of actions to perform. Since high-level planning and problem-solving are highly variable and unpredictable, KLM restricts its focus to the linear execution of physical acts (keystrokes, mouse movements) and the brief mental pauses required to verify those routine actions.
